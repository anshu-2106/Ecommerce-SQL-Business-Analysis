# Ecommerce SQL Business Analysis

## Project Overview

Performed business analysis using SQL to evaluate revenue performance, customer behavior, payment trends, product ranking, and sales patterns.

## Tools Used

- MySQL Workbench
- SQL
- Business Analytics

## SQL Concepts Used

- Aggregations
- GROUP BY
- ORDER BY
- CASE WHEN
- HAVING
- CTEs
- Window Functions
- RANK()
- Date Functions

## Key Analysis Performed

- Total Revenue Calculation
- Revenue by Category
- Customer Segmentation
- High Spending Customer Analysis
- Payment Method Analysis
- Product Ranking by Category
- Monthly Revenue Trend

## Key Insights

- Electronics generated highest overall revenue.
- iPhone 14 ranked highest within Electronics.
- Digital payment methods contributed strongest revenue.
- Month 11 recorded peak revenue performance.
